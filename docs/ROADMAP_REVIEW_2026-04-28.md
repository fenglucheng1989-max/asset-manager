# Roadmap Review 2026-04-28

## 资产趋势与首页

- 首页保留“资产趋势”入口，但不在首页展示相对最近快照的弱意义统计。
- 首页“资产配置”能力由“我的账户”列表承接，保留金额占比辅助信息，避免与资产趋势详情重复。
- 资产趋势页保留折线图、悬浮点选提示和快照分页加载；去掉与折线图重复的“资产变化”块。
- 资产趋势页账户排行补充占总额比例，作为当前资产结构表达。

## 账户录入

- 新增账户页按“资产 / 负债”分标签录入。
- 账户类型选择器增加容器层级与宽度约束，降低 H5 下被遮盖的问题。
- 负债归属由顶部标签表达，不再使用“统计设置选择是否为负债”这类生硬入口。

## 记一笔与账本

- 底部“记账”改为“账本”，列表页文案从“流水”调整为“收支/记录”，降低术语感。
- “记一笔”保持独立页面。
- 记一笔时支持就地新增分类，不再要求用户先跳到分类管理页。
- 账本页筛选采用轻量顶部工具区，周期模式收纳到日期标题中。
- 类型筛选与分类筛选保持独立；选择“转账”时分类显示为禁用态“无分类”。
- 月度预算在账本页展示摘要状态，包括已用、总额、剩余或超支、进度和预警状态。
- 收支结构合并为一张环形图卡片，通过“支出 / 收入”切换。
