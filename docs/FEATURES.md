# 当前功能清单

## 已实现功能

### 用户端 H5 / App 基础能力

- 用户注册、登录、退出登录。
- JWT 登录态本地保存。
- 登录过期后清理本地 token，并跳转到“我的”页。
- “我的”页为账户中心，展示头像首字母、用户名、邮箱、注册日期、净资产摘要和功能入口。
- “我的”页支持设置默认本位币，默认值为人民币 `CNY`，新增账户时优先使用该币种。
- “我的”页提供资产趋势、账户管理、快照管理入口；净资产摘要可返回资产首页。
- “我的”页预留主题设置、导出数据、备份与同步、清空所有数据、修改密码、生物识别解锁、注销账号、用户协议、隐私政策和意见反馈入口；未接入后端能力前以提示方式展示。
- H5 桌面预览使用手机宽度居中展示。
- H5 启动脚本已切换 UTF-8，中文提示可正常显示。
- 首页进入和返回时自动刷新数据，加载时展示轻量状态提示。
- 首页不提供下拉刷新，避免与列表滚动、长按排序产生手势冲突。
- App 首屏有保护逻辑，接口或状态读取失败时尽量避免白屏。

### 资产账户

- 查询账户列表。
- 查看账户详情。
- 新增账户。
- 编辑账户。
- 删除账户。
- 归档账户，归档后不在用户端首页展示，也不参与总资产、总负债和净资产统计。
- 账户详情页支持独立调整当前余额，无需进入完整设置表单。
- 查看账户余额变化记录，包括创建账户、手动调整余额、流水记账、调整前余额、调整后余额和变化金额，记录以金额流转形式展示。
- 后端提供账户排序接口；首页按用户自定义顺序展示“我的账户”，新增账户默认插入列表最前。
- 添加账户入口位于账户列表底部，账户列表标题区不放置额外操作按钮。
- 支持账户名称、类型、币种、折算汇率、余额、资产/负债归属、是否计入净资产、颜色、备注。
- 账户设置页先选择资产或负债，再录入账户类型、金额与币种、统计展示信息。
- 账户表单支持名称长度提示、金额格式校验、备注长度提示和离开确认。
- 非 CNY 账户按账户级折算汇率折算为 CNY 后参与总览、快照、占比和趋势页账户金额排行。

当前账户类型：

- `CASH`：现金
- `BANK`：银行卡
- `CREDIT_CARD`：信用卡
- `E_WALLET`：电子钱包
- `INVESTMENT`：投资理财
- `LOAN`：贷款
- `REAL_ESTATE`：房产
- `OTHER_ASSET`：其他资产
- `OTHER_LIABILITY`：其他负债

### 资产总览

- 展示净资产、总资产、总负债。
- 首页净资产卡片可点击进入资产趋势页，不再单独放置资产趋势 card。
- 首页展示“我的账户”管理列表，保留账户占比进度作为辅助信息，但不按金额强行排序。
- 首页账户支持长按拖动排序，松开后保存顺序；左滑可删除账户。
- 资产趋势页支持按日、月、年切换查看净资产折线走势，点击折线点可查看当前金额和环比变化。
- 资产趋势页展示账户金额排行，默认按折合 CNY 金额从高到低排列，并展示占比。
- 资产趋势页快照记录按页加载，避免历史数据过多时一次性渲染。
- 支持手动记录今日资产快照，同一天重复记录会更新当天快照。
- 首页避免重复页面标题、常驻刷新按钮、排序说明等解释性 UI，占用空间的说明只在实际操作时短提示。
- 总资产只统计 `include_in_total=true`、`archived=false` 且 `is_liability=false` 的账户。
- 总负债只统计 `include_in_total=true`、`archived=false` 且 `is_liability=true` 的账户。
- 净资产 = 总资产 - 总负债。
- 不计入净资产的账户仍显示在账户列表中。
- 多币种账户以账户原币种保存余额，同时按折算汇率换算为 CNY 展示汇总金额。

### 账本与记一笔

- 底部导航提供独立“账本”入口，新增和编辑记录使用“记一笔”页面。
- 账本页展示所选周期的收支净额、收入合计、支出合计和明细。
- 账本页支持本周、本月、今年与前后周期切换；周期模式收纳在日期标题内，不常驻大块筛选控件。
- 账本页支持类型和分类筛选。类型筛选与分类筛选默认互不重置；当类型选择“转账”时，分类筛选显示为禁用态“无分类”，查询时忽略分类条件。
- 账本页展示月度预算摘要，包括已用、预算总额、剩余或超支、进度条和正常/预警/超支状态；点击可进入预算设置。
- 账本页展示所选周期的收支结构环形图，支出/收入在同一卡片内切换展示。
- 支持新增收入、支出、转账记录。
- 支持点击记录进入编辑。
- 支持选择账户、转入账户、分类、金额、标签和备注。
- 收入流水自动增加账户余额。
- 支出流水自动扣减账户余额。
- 转账流水自动扣减转出账户并增加转入账户，总资产保持不变。
- 长按记录可删除，删除后自动撤销该记录对账户余额的影响。
- 记录变更会同步写入账户余额变化记录，账户详情页可追溯。
- 支持在记一笔时就地新增自定义收入和支出分类，系统默认分类不可编辑。
- 支持月度预算、分类预算、预算使用率和预警展示。
- 支持月度收入、支出、结余、收支分类占比和日趋势报表。

### 后台管理端

- 管理员登录。
- 平台数据概览。
- 用户列表、用户名/邮箱搜索、用户资产摘要。
- 用户角色调整。
- 账户列表、按用户 ID/账户名/账户类型筛选。
- 编辑和删除账户。
- 流水管理，支持全平台月度收支总览、流水筛选和流水明细查看。

### 后端能力

- 认证接口：注册、登录。
- 用户接口：查询个人资料、设置默认本位币；个人资料包含用户 ID、用户名、邮箱、角色、默认本位币和注册时间。
- 资产接口：总览、列表、详情、新增、编辑、归档、删除、排序、账户余额历史、创建快照、查询快照。
- 流水接口：分类列表、自定义分类维护、流水筛选、新增流水、编辑流水、删除流水、预算维护、月度报表。
- 后台接口：平台概览、用户管理、账户管理。
- 系统支持每日定时生成资产快照。
- Spring Security + JWT 鉴权。
- 管理接口要求 `ADMIN` 角色。
- 本地 `local` profile 自动创建或修正管理员账号。
- 资产总览支持 Redis 缓存；Redis 不可用时降级为直接查数据库。
- 常见 API 响应消息已中文化。
- 后端已补充认证、用户默认币种、账户 CRUD、排序、账户归档、账户余额历史、多币种折算、资产总览、资产快照、记账流水、流水筛选、自定义分类、预算和报表集成测试。

### 存储与运行

- 默认 H2 文件存储。
- `local` profile 使用 H2 文件存储。
- `postgres` profile 保留 PostgreSQL 配置。
- Docker Compose 保留 PostgreSQL 服务，但通过 compose profile 控制是否启动。
- 用户端 API 地址通过 `frontend/src/config/api.js` 和 Vite 环境变量统一管理。

## 已提供但尚未完整产品化

- 后台管理已有基础管理能力，但尚未做分页、操作日志、批量操作。
- 还款、收益专用交互、附件、票据图片和导入能力暂不纳入近期规划，后续按真实使用价值再评估。

## 当前边界

- 当前版本不自动采集银行、支付平台或证券账户数据。
- 当前版本不支持家庭成员共享资产。
- Android 包当前用于测试，不作为正式应用市场上架包。
- H5 本地代理只用于开发联调，正式部署应由 Nginx 或网关统一代理 API。
